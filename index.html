Pasta Alfredo2

Welcome to the predecessor to the exceptional adventure game Pasta Alfredos Adventure.
This game was made by David Persson using HTML & bootstrap.
You play this game by reading the story, then you choose which actions you want to take. There is currently three 'happy' endings in the game.
If you die or reach the end of a certain storyline you will get the choice to restart, allowing you to replay the game following a different route.


WARNING!
This game contains criminal activites and disturbing content that might be seen as offensive or disturbing.
Play at own discretion.


Press START to begin the adventure.
